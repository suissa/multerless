# Relatório de Atualização da Landing Page

## O que foi feito

A página de apresentação (landing page) localizada em `landing/` foi atualizada para refletir a identidade visual solicitada:

1.  **Atualização do Background**:
    - O fundo da página foi alterado para a cor sólida `#0d1117` (GitHub Dark Dimmed), substituindo o gradiente anterior. Alteração realizada em `landing/src/style.css`.

2.  **Inserção do Logo**:
    - O emoji 📤 no cabeçalho foi substituído pelo logo oficial do projeto (`https://i.imgur.com/nE0RRoo.png`).
    - O logo foi ajustado com efeito de sombra (`drop-shadow`) para manter o destaque sobre o fundo escuro.
    - O texto do título duplicado foi ocultado, mantendo a limpeza visual.

3.  **Documentação**:
    - O `CHANGELOG.md` foi atualizado com as mudanças de interface.

## Por que foi feito

Para alinhar a apresentação visual do projeto com a identidade da marca solicitada pelo usuário, utilizando uma estética "Dark Mode" moderna e profissional, consistente com o ecossistema de desenvolvimento (GitHub).

## Detalhes Técnicos

- **Arquivos modificados**:
  - `landing/src/style.css`
  - `landing/src/main.js`
  - `CHANGELOG.md`
- **Stack**: Vite + Vanilla JS + Tailwind CSS (via PostCSS).
